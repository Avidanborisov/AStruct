void *(* ANew)(Type,...)

Create a new data structure.

Parameters
TypeData structure type
...Additional/optional arguments
Returns
Pointer to the new data structure or NULL on error (such as missing or invalid arguments)

Create a new data structure. This function accepts an AStruct type such as AList and creates a new instance of it with all the default values it should have.

Each data structure may require additional or optional arguments which are supposed to be passed too, after the type name. To pass an argument, you must as well pass all the arguments before that argument, even if they're optional.

Examples of usage:

AList* list = AStruct->ANew(AList); // Create a new list
AVector* vector = AStruct->ANew(AVector, 100); // Create a new vector with initial capacity of 100 items
void *(* ADup)(Data,...)

Duplicate data.

Parameters
DataA lvalue expression
...Optional size argument
Returns
Pointer to malloced() memory with the same value as Data.

Convenience function for duplicating a variable. This function accepts any variable (which has to be a lvalue) and duplicates it to new memory allocted by malloc().

By default this function duplicates sizeof(Data) bytes, but if passed another argument after Data, that argument will be considered as the number of bytes to duplicate.

Examples of usage:

int foo = 15;
int* bar = AStruct->ADup(foo); // bar points to dynamically allocted memory. *bar == foo
char text[] = "Hello!";
char* copy = AStruct->ADup(*text, sizeof text); // same effect as calling strdup(text)
