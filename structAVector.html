Detailed Description

Dynamic array.

This data structure is a generic dynamic array which expands on its own. Use a vector whenever you need an array of unknown size which expands on its own. Dynamic arrays have fast insertations and removals from the end, and random access is constant.

The arguments passed to AStruct->ANew() to create a new vector are:

AStruct->ANew(AVector, int capacity)
Parameters
[opt]capacity= Optional argument to specify the initial capacity of the vector

Examples of creating a new vector:

AStack* vector = AStruct->ANew(AVector);
// create a vector with initial capacity of 100 items
AStack* vector = AStruct->ANew(AVector, 100);
