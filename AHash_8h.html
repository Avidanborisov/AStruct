typedef size_t(* AHashFunc)(const void *)

Hash function type.

The hash function gets a key pointer and returns an unsigned integer hash value of the key. The hash function should uniformly distribute the hash values for each key to minimize collisions.

It is advised not to roll your own hash functions but to use AHash->hash() to implement your hash functions.

size_t(* hash)(const void *key, size_t size)

Generic hash function.

Parameters
keyPointer to data
sizeSize of the data in bytes
Returns
Hash value of the data

Hash the data pointed to by key with the given size of bytes. This generic hash function is and should be used to implement hash function for various data types.

Currently the hash function is MurmurHash2.

const __AHash* AHash

Hash functions.

AHash is a pointer identifier which provides you with hash functions (suitable for data structures such as AHashtable).

AHash->hash() is a generic hash function which should be used to implement hash functions for custom types not provided by AHash.

AHash also provides you with built-in hash functions for convenience:
